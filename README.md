# Python
## 📊 NFHS-5 Socio-Economic Data Analysis

### Project Overview
This project provides a comprehensive visual analysis of the **National Family Health Survey (NFHS-5)** data for India. The goal is to uncover relationships between women's empowerment indicators (literacy, internet usage) and critical health outcomes (fertility rates, institutional births) across different states and urban/rural divides.

### 🔍 Key Insights & Visualizations
- **Correlation Heatmaps:** Identifying strong links between education levels and reproductive health choices.
- **Urban vs. Rural Comparison:** Boxplots and bar charts highlighting the developmental gap in digital and traditional literacy.
- **Multi-Variable Distributions:** Pairplots showing how states cluster based on socio-economic performance.
- **Demographic Breakdown:** Pie charts showing household survey distributions.

### 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** - `Pandas` for data cleaning and manipulation.
  - `Seaborn` & `Matplotlib` for high-quality statistical visualizations.
  - `NumPy` for numerical processing.

### 📈 How to Interpret the Results
The analysis reveals that states with higher female literacy typically exhibit higher internet penetration and lower Total Fertility Rates (TFR). The "Urban-Rural" toggle in the visualizations further highlights the geographical disparities in health infrastructure and digital adoption.
